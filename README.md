# BA Unified Helper

BA Unified Helper is a RuneLite plugin for Barbarian Assault that combines calls, timers, enemy counts, cannon ammo display, ground item labels, and role-based highlights into one configurable plugin.

## Features

- Current call display
- Correct call horn highlight
- Call-change timer
- Wave number and wave timer
- Game timer / completion duration
- Enemy counts
- Cannon ammo text above cannon
- Configurable BA HUD overlay
- Configurable ground item labels and highlights
- Role-specific NPC highlights

## Role Helpers

### Attacker

- Highlight Penance Fighters and Rangers
- Show the current attacker call
- Highlight the correct combat style, ammo, or spell where detectable

### Collector

- Highlight the correct egg color
- Label egg colors
- Highlight correct eggs in inventory
- Show cannon ammo

### Defender

- Highlight Penance Runners
- Highlight the correct food: Tofu, Crackers, or Worms
- Label relevant dropped food

### Healer

- Highlight Penance Healers
- Highlight the correct poisoned food
- Show healer count
- Show teammate health information where available

## Scope

BA Unified Helper is focused on overlays, labels, timers, counters, and highlights.
